1. Wymienić oraz opisz dwie części selektora,


Selektor – określa, do jakiego elementu HTML mają zostać zastosowane style. Może to być np. p, .klasa, #id.
Deklaracja – zawiera cechy i wartości w nawiasach klamrowych {}.

2. Znaczenie nawisów klamrowym {...},


Nawiasy klamrowe służą do grupowania deklaracji związanych z danym selektorem. Wszystkie właściwości CSS dla wybranego elementu muszą być umieszczone wewnątrz tych nawiasów.

3. Opisz cechy,

Cechy (ang. properties) to właściwości stylizujące element HTML.

Przykładowe cechy:

color – kolor tekstu
font-size – rozmiar czcionki
background-color – kolor tła
border – obramowanie elementu
margin – marginesy zewnętrzne

4. Opisz wartości


Wartości to konkretne ustawienia dla cech CSS. Mogą być liczbowe, tekstowe lub predefiniowane.

Przykład:
<style>
     p {
           color: red; /* Kolor tekstu */
           font-size: 16px; /* Rozmiar czcionki */
           margin: 10px; /* Marginesy */
           display: flex; /* Typ wyświetlania */
    }
</style>

5. Kiedy stosujemy znak średnika (;), kiedy nie jest konieczny


Średnik jest obowiązkowy po każdej deklaracji, gdy mamy więcej niż jedną linię wewnątrz {}.
Nie jest konieczny w przypadku pojedynczej deklaracji.
